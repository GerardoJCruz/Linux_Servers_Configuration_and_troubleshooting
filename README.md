# Multi-Server Linux Deployment & Enterprise Ticketing Simulation. 

## Project Description:
This project is designed to mock an enterprise production environment, demostrating a real world workflow for configuring, managing, and automating a pair of Linux servers. 
The project consist in a list of production style tickets to  mirror a lifecycle of a System Administrator or Junior DevOps Administrator. 

## Key Features:
- Production Style System's Tickets: Multi-Stage task tracking representing typical infrastructure requests. 
- Automated Configurations: Automated versions for the repetitive tasks. 
- Documented Troubleshooting: Document any troubleshooting encountered during the development of this project. 

## Some of Concepts touch in this project are: 
- Networking configurations.  
- Firewall permissions and security. 
- Logical Volume Management. 
- Users control. 
- System scheduling. 
- SELinux policies and context remedation. 
- Containers. 
- Secure connections.

## Servers:
servera.lab.local: REHL9.7 Server - Main Server
serverb.lab.local: AlmaLinux9 Server - Secondary Server
